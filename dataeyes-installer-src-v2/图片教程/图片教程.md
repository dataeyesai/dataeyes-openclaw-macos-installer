# DataEyes × OpenClaw 安装图片教程

按下面顺序看图：

1. `01.png`：第一次打开前，先看桌面安装包内容
2. `02.png`：双击 `双击开始安装.command`
3. `03.png`：如果系统提示安全限制，到“系统设置 / 隐私与安全性 / 安全性”里点“仍要打开”
4. `04.png`：重新打开安装程序
5. `05.png`：等待 OpenClaw 基础环境安装完成
6. `06.png`：按提示输入 DataEyes API Key
7. `07.png`：等待安装器自动写入 DataEyes 配置并启动 Gateway
8. `08.png`：安装完成后会自动打开本机 OpenClaw 控制台

## 说明
- 默认模型：`dataeyes/gpt-5.4`
- 自动回退：`dataeyes/gemini-3.1-pro-preview-customtools`、`dataeyes/claude-opus-4-6`
- 用户只需要双击 `双击开始安装.command`
- 其他文件都在 `内部文件/`，不用手动点
